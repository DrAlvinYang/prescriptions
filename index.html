Getting Started

This tool streamlines creating discharge prescriptions within the ED.

Workflow

  1. Provider Information: Input your name and CPSO number.
  2. Location Selection: Set the work location to appear on the script.
  3. Select Prescriptions: Use the search bar or specialty folders to find and add prescriptions to the cart.
  4. Print: Print the prescriptions in cart.

Search Features

  • Advanced Search: You can search by indications (e.g. ped otitis media), brand names (e.g. keflex), and dosings (e.g., 400mg, PO, q4h, 7 day).
  • Specialty Folders: Prescriptions are organized by specialty and subdivided into Adult and Pediatric populations.
  • Weight-Based Dosing: Pediatric doses are calculated automatically when weight is provided. Doses are capped at the maximum standard dose.
  • Curated Database: This tool contains a curated set of ~400 common ED prescriptions; it is not an exhaustive pharmacopeia.

Disclaimer and Limitations

  • This tool is for physician use only.
  • It is not a substitute for clinical judgment.
  • Always verify indications, doses, and required renal or hepatic adjustments.
  • The system does not check for patient allergies, drug interactions, or safety in pregnancy and lactation.
  • Content may be incomplete or outdated.
